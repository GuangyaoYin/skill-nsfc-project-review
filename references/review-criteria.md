# NSFC Review Criteria

## Three Required Review Items

1. 是否面向经济社会发展需要或国家需求背后的基础科学问题：判断项目是否从实际需求中凝练出清楚、可研究、可检验的基础科学问题，而不是只停留在应用背景或工程任务描述。
2. 科学问题的创新性与预期成果的科学价值：判断新意来自科学假说、机制认识、数据约束、方法体系、研究对象还是交叉视角，并判断预期成果是否能形成学术增量。
3. 研究基础与可行性，以及完善建议：判断申请人的前期积累、关键数据/样品/平台、技术路线、风险控制和替代方案是否支撑项目目标。

## Grade Criteria

### A / 优 / 优先资助

Use when the proposal has a clear and important scientific question, strong target-oriented basic research attributes, substantive innovation, coherent technical route, solid applicant foundation, and convincing feasibility. Comments should be positive overall and include limited suggestions for refinement.

### B / 良 / 可资助 or 优先资助

Use when the proposal is meaningful and generally feasible, with some innovation and adequate foundation, but has limitations in problem refinement, innovation distinctiveness, validation design, route closure, or expected scientific output. Comments should support funding conditionally or comparatively, while naming real weaknesses.

### C / 中 / 不予资助

Use when the proposal has some research value but weak competitiveness. Typical reasons include diffuse scientific questions, limited innovation, insufficient logical linkage between objectives and methods, weak validation, or inadequate prior foundation for key tasks.

### D / 差 / 不予资助

Use when the basic scientific question is unclear or not established, innovation is weak, research contents are piled up, the technical route cannot support objectives, or key data/method/foundation conditions are missing.

## Common Wording Patterns

Positive opening:

"该项目围绕……中的……问题展开，具有明确的国家需求背景和一定的基础科学研究属性。申请书试图从……角度揭示……机制/规律/方法基础，选题具有较好的科学意义和应用牵引价值。"

Balanced limitation:

"需要指出的是，申请书对……的科学问题凝练仍可进一步加强，部分研究内容与核心假说之间的逻辑关系尚不够紧密，预期成果的科学内涵也有进一步聚焦空间。"

Innovation judgment:

"项目的创新性主要体现在……。与已有研究相比，其潜在增量在于……，但申请书仍需进一步说明……与现有方法/认识之间的实质区别。"

Feasibility judgment:

"申请人在……方面已有一定研究积累，前期工作为本项目实施提供了基础。研究方案总体较完整，但……环节仍存在不确定性，建议进一步明确关键数据来源、参数约束、验证方案和风险替代路径。"

Non-funding judgment:

"总体看，项目具有一定研究意义，但目前申请书对核心基础科学问题的凝练不足，创新点与已有研究的区别不够清晰，技术路线对预期目标的支撑仍显薄弱，因此在本批次中的资助竞争力有限。"

## Long-Comment Expansion Guide

For each 800-1000 character item, write one coherent paragraph or two long paragraphs. Expand using evidence from the proposal:

- background and need
- core scientific question
- existing research gap
- proposed hypothesis or mechanism
- methods and data that address the question
- expected scientific contribution
- limitations and suggestions

Avoid repeating the same sentence across the three items. Do not pad with empty praise; use proposal-specific evidence.
